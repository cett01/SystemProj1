For Part 2;
All necessary commands to run the program is included in "script.sh" file.
Therefore, in order to run, please change the access and permission rights
of "script.sh" file by writing 
	chmod +x script.sh
in the command prompt.
After that just run script file via 
	./script.sh
,this script will create all needed files such as ".lis", ".o" for asm 
and ".o" object file for the code
and enjoy :)
